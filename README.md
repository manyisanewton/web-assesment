# web-assesment

Responsive frontend assessment recreation using:
- HTML
- CSS
- JavaScript
- Bootstrap 5

## Features
- Responsive, dashboard-style layout using Bootstrap grid/components
- Membership sections with reveal/hide interactivity
- Navbar menu toggle and modal interactions
- Simple email validation
- Organized structure: `index.html`, `styles.css`, `script.js`

## Run locally
Open `index.html` in your browser.
